## 설치된 의존성

### UI 컴포넌트 (자동 설치 완료)

**shadcn/ui** — `src/components/ui/` 위치
- `button`, `card`, `dialog`, `tabs`, `avatar`

**Magic UI** — 같은 위치
- `shimmer-button` (CTA 골드 효과)
- `blur-fade` (등장 페이드)
- `number-ticker` (신뢰 지표 카운트업)
- `marquee` (후기 무한 슬라이드)

### Aceternity UI — ⚠️ 수동 작성 대상

`ui.aceternity.com/r/*.json` registry가 2026-05-12 기준 모두 404. 자동 설치 불가.
필요한 컴포넌트는 [공식 사이트](https://ui.aceternity.com/components) 코드 복사 후 `src/components/ui/` 에 직접 생성.

대상 (필요 시 작성):
- text-generate-effect (히어로 헤드라인, GSAP SplitText로 대체 가능)
- spotlight (히어로 라이팅)
- compare (Before/After 슬라이더 — **WOW A 갤러리에 필수**)
- focus-cards (포트폴리오 그리드)
- sticky-scroll-reveal (시공 과정)
- tracing-beam (스크롤 라인)

대체 라이브러리 옵션: `react-compare-slider` (Compare 한정)

## 디자인 토큰

`src/app/globals.css` 에 정의됨:

| 토큰 | 값 | 용도 |
|------|------|------|
| `--brand-gold` | `#d4af37` | CTA, 강조 |
| `--brand-bronze` | `#b8772a` | 그라데이션 종단색 |
| `--brand-black` | `#0a0a0a` | 배경 |
| `--gradient-gold` | `linear-gradient(135deg, #d4af37, #b8772a)` | Primary CTA · 강조 카드 배경 |
| `--radius` | `0.625rem` | 기본 모서리 |
| `--font-sans` | Pretendard Variable | 본문 (한국어 + 일반 텍스트) |
| `--font-display` | Anton, Bebas Neue, Impact | **영문 헤딩 / 큰 숫자 / 영문 라벨 (다이나믹 톤)** |
| `--background` / `--foreground` | shadcn neutral | 기본 |

Tailwind 클래스: `bg-brand-black`, `text-brand-gold`, `bg-brand-bronze`, **`font-display`** (영문 헤딩용).
**그라데이션 사용**: `className="bg-gradient-to-br from-brand-gold to-brand-bronze"` 또는 `style={{ background: "var(--gradient-gold)" }}`.
적용 대상: Primary CTA(Nav 견적 문의 / Hero 상담 예약 / Footer 카톡 상담), Pricing **Standard** 카드 배경.

### 폰트 로드 (`layout.tsx`)

```tsx
import { Anton } from "next/font/google";
import "pretendard/dist/web/variable/pretendardvariable.css";

const anton = Anton({ subsets: ["latin"], weight: "400", variable: "--font-display", display: "swap" });

export default function RootLayout({ children }) {
  return (
    <html lang="ko" className={anton.variable}>
      <body>{children}</body>
    </html>
  );
}
```

영문 헤딩(BLACKOUT GLOSS, Services., Pricing., 큰 숫자 1,247+, BEFORE / AFTER, PROCESS 등)에 `className="font-display"` 사용.
한국어는 기본 Pretendard 그대로.
