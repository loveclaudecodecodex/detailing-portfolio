## 필수 영상 (2개)

### 1. `hero-bg.mp4` — 히어로 배경 반복재생

| 항목 | 값 |
|------|------|
| **길이** | 8~12초 (무한 루프) |
| **PC 해상도** | 1920×1080 (16:9) |
| **모바일 해상도** | 1080×1920 (9:16) — 별도 `hero-bg-mobile.mp4` |
| **포맷** | MP4 (H.264), AAC 오디오 제거 |
| **용량** | 5MB 이하 (PC), 3MB 이하 (모바일) |
| **프레임** | 24/30fps |
| **컬러 톤** | 다크 (블랙 + 골드 액센트, 야간 도심 / 스튜디오) |

**촬영/연출 가이드:**
- 차량 클로즈업 (보디 라인, 광택 반사, 휠 디테일)
- 슬로우 모션 (실속도 1/4)
- 카메라 워크: 슬라이드, 돌리, 짐벌 워킹
- 시작·끝 프레임을 동일하게 → 루프 이질감 없게
- 텍스트가 위에 얹히므로 **중앙 영역은 비교적 단순한 구도** 권장

**시작 후보:**
- Veo (AI): 프롬프트 예시 "Cinematic close-up of a glossy black BMW 5 series in a dark studio, slow motion droplets, golden rim light, 4K"
- Pexels: "car detailing", "luxury car", "black car closeup"

### 2. `process-timelapse.mp4` — WOW C 시공 타임랩스

| 항목 | 값 |
|------|------|
| **길이** | 30초 (스크롤 200% 분량과 매핑) |
| **해상도** | 1920×1080 |
| **포맷** | MP4 (H.264) |
| **용량** | 10MB 이하 |
| **프레임** | 30fps |

**구성 (12시간 시공 → 30초 압축):**
- 0~5초: 입고 + 외관 점검
- 5~12초: 정밀 세정 (철분 제거 / 클레이바)
- 12~22초: 코팅/PPF 시공 (가장 비중 ↑)
- 22~27초: 광택 마무리
- 27~30초: 완성 차량 360 회전

**주의:**
- 카메라 위치 고정 (타임랩스 정통 방식) 또는 점진적 줌
- 스크롤로 재생 위치를 매핑하므로 **연속적·일정한 변화율** 권장 (점프 컷 X)

| `wow-before.jpg` | 1920×1080 | 시공 전 — 먼지·물때·세차 안한 상태 |
| `wow-after.jpg` | 1920×1080 | 시공 후 — 광택·반사·코팅 완료 |

**필수 조건:**
- 두 사진의 **차종 / 각도 / 광원 / 카메라 위치 동일**
- 다크 컬러 차량 권장 (광택 차이 극대)
- 배경 동일 (배경 변하면 변신 효과 깨짐)

**제작 방법:**
1. 동일 차량을 더러운 상태 + 시공 후 각각 촬영 (가장 정확)
2. AI 생성: "before" 프롬프트 + "after" 프롬프트 + Seed 고정
3. Pexels 한 장 + Photoshop으로 "before" 만들기 (가장 빠름)

## 형식·최적화 규칙

- **이미지**: JPG 85% (사진) / PNG (투명) / SVG (아이콘)
- **자동 변환**: Next.js `<Image>` 가 WebP/AVIF로 자동 변환 — 원본 JPG/PNG로 저장
- **영상**: H.264 MP4 self-host. 향후 트래픽 늘면 Cloudflare Stream
- **명명**: 영문 소문자 + 하이픈 + 섹션-순번-상태 (예: `portfolio-01-before.jpg`)
- **시안용**: `_draft` 접미사 (예: `hero-bg_draft.mp4`)
