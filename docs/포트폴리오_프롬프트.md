## 🎨 갤러리 전체 일관성 가이드

6쌍을 나란히 놓고 봤을 때:
- 배경 톤이 모두 다크 차콜로 통일되어야 함
- 광원 색온도가 모두 골든 워옴 톤으로 일치
- 바닥(폴리시드 콘크리트) 반사 양상이 비슷해야 함
- 차의 화면 점유율이 60~75% 범위로 비슷

흩어진 인상이면 **가장 좋은 1쌍을 reference로** 나머지 재생성.
